> Project Overview

This project is a frontend landing page that includes:

A responsive navigation bar

A hero section with call-to-action buttons

A feature section highlighting multiple features

Clean layout using Flexbox

Custom UI elements like cards and circular icons

It is ideal for practicing layout design, positioning, and UI structuring using pure HTML and CSS.

> Technologies Used

HTML5

CSS3

Flexbox

Custom Styling & Layout Design

Hover Effects

> Features

. Modern navigation bar

. Hero section with heading, description, and CTA buttons

. Image card with shadow effect

. Feature section with multiple feature cards

. Clean and minimal UI

. No JavaScript used (pure HTML & CSS)
